Q1 : How does javascript work?

Ans:All programming language is in a human-readable format, but our computer does not understand this, understand machine code which We know as ones and zeroes. A browser has three main programs, the first program is the dom parser and which takes our HTML code and converts it into a structure that we visually see, the text parser is the CSS parser which makes sure our document is rendered correctly. The layout engine or rendering engine inside of a browser takes both HTML and CSS and ultimately renders the page. Javascript engine(eg: V8 engine, spider monkey)takes the javascript syntax in human-readable form and converts it into machine code so that our computer can understand our instruction, which is called client-side. javascript engine interprets this or it's going to have compilation involved there, such as JIT(just in time) compiler, which takes human-readable instructions that we execute at that specific time and converts those instructions very quickly. Javascript works this way.
